## Workflow
When told to work on a Linear ticket:
1. Look up the Linear ticket.
2. Read any subtasks or child issues.
3. Propose any new subtasks that should be added.
4. If the user accepts the proposed subtasks, add them to Linear.
5. Begin implementing what the ticket says to do.
